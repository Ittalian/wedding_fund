# 目標達成のための提案機能 実装完了

目標達成に向けた資金計画で「不足」が生じた場合に、どのように見直せばよいかを提案する機能を実装しました。

## 実装内容
- **[MODIFY] [app_state_provider.dart](file:///Users/itta/dev/dart/wedding_fund/lib/providers/app_state_provider.dart)**
  - 出費額計算（`FinancialCalculation`）内に、不足を解消するための2つのシミュレーションロジックを追加しました。
  - **減額提案 (`reductionSuggestions`)**: すべての費用項目（および目標貯金額）に対して、「その項目単体でいくらまで減額すれば達成可能か」を二分探索で計算します。
  - **時期見直し提案 (`delaySuggestions`)**: 現在の月々の固定費設定を維持したまま、不足する費用項目を「いつまで延長すれば無理なく払えるか」を計算します。
- **[MODIFY] [home_screen.dart](file:///Users/itta/dev/dart/wedding_fund/lib/screens/home_screen.dart)**
  - 「出費額計算」モードの画面に、「不足を解消するための提案」セクションを新設しました。
  - 減額提案と時期見直し提案をわかりやすく箇条書きで表示するようにしました。
  - 提案項目が多くなっても画面がはみ出さないように、画面全体をスクロール可能（`SingleChildScrollView`）に修正しました。

## 確認事項
計算結果が不足と判定された際、赤い警告画面の下に以下のような具体的な提案が表示されるようになっています。

1. **費用の減額見直し**
   - 例: `・結婚式 を ¥1,500,000 以下にする`
   - ※単体で減らしても達成不可能な場合は「複数の項目を合わせて減額するか、時期の延長をご検討ください」と表示されます。
2. **費用の時期見直し**
   - 例: `・指輪・新婚旅行 を 2026年12月 まで伸ばす`
   - これにより、金額を減らさずに時期をずらすという選択肢も比較しやすくなりました。
